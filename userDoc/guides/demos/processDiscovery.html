Process Discovery

In this example, we demonstrate how to apply process discovery in PM4KNIME.

An example workflow with PM4KNIME is shown below:

The following steps will guide you to accomplish the task and discover a Petri net from an XES file.

  1. Drag the node XES Reader from the Node Repository into your workflow.

  2. Configure the XES Reader with the path to your XES file, then click "OK" to close the dialog.

  3. After executing the node, you can right-click and select "Event Log" to open a view for inspecting the event log.

  4. Drag the node Inductive Miner into your workflow and connect its input port with the output port of the XES Reader. Configure the settings with the following example values, then Click "OK".

  5. Execute the Inductive Miner.
  6. After executing the node, you can right-click and select "Process Tree" to open a view for inspecting the generated process tree.

  7. Optional: you can use the node Process Tree to Petri Net Converter to transform the generated process tree into a Petri net.
  8. Optional: you can export the generated Petri net with node Petrinet Writer.
